Bloom

Flower Shop Catalog

Bloom is a dedicated mobile app that let the user customize orders and speed up the ordering process.

Challenge

In certain times of the year flower shops are overwhelmed and it is difficult to manage orders for them. Long queues are formed at shops and if bouquets are not ordered in advance results sometimes are not what clients wait.

Solution

Design a flower shop app that allows users to easily order and pick-up orders through the application and allows a high degree of customization whenever the user wants.

My role

UX designer designing Bloom's app from conception to delivery.

Date

January 2023 to April 2023

Process

Low fidelity prototype

Key findings & insights

Key findings

  • Some parts of the design need to be more intuitive.
  • Some more security needs to be added to the design so the users feel at ease with sensible data.
  • Workers functionality is not covered yet and needs to be extended.
  • People not used to technologies and older people need the app to be easier so they can understand it better.

Key insights

  • Adapt the app to be easier to use for old people and people with visual impairments.
  • Inform users of how their data is used and protected and how they can delete it if needed. They need to be informed about the benefits of having a user too. This can be added to the user section.
  • Facilitate the app usage including more design cues and replacing the scrolling layout by full page layout. Some icons will be replaced by other more understandable.
  • Add more functionality to help flower shop workers.

High fidelity prototype

CUSTOM & AVAILABILITY

Users customize their orders and see flowers availability before ordering.

EMERGENCY CONTACT

Users add a number to contact the person to deliver in case flower workers can't find the person.

ORDER TRACKING

Users can track their order, and if consented, workers can track the person to deliver location.

FAVORITE PRODUCTS

Users can save their favorite products or orders to repeat them whenever they want and make a quick purchase order.
